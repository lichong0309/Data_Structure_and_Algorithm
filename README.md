## 本仓库

本仓库使用Python实现主流的数据结构和算法，包括链表，队列，树，图，排序，以及LeetCode hot100的代码实现和解题思路。

### 使用方法




> pip3 install -r requirement.txt					安装相应的Python包
>
> tree > tree.txt 								    得到仓库文件目录
